로컬 환경 전용 API

이 API는 TIA Portal이 설치된 로컬 PC에서만 동작합니다. 아래 조건을 모두 충족해야 합니다:

  • Siemens TIA Portal V20이 로컬 PC에 설치되어 있어야 합니다
  • .NET Framework 4.8 런타임이 필요합니다
  • Windows 사용자가 'Siemens TIA Openness' 그룹에 등록되어 있어야 합니다
  • TiaApiServer.exe를 로컬에서 실행한 후, TIA Portal을 시작하고 /api/connect를 호출하여 연결해야 합니다
  • 이 문서의 "Try it out" 기능은 로컬에서 API 서버가 실행 중일 때만 동작합니다

Siemens Engineering DLL

API 서버 실행에 필요한 Siemens.Engineering.dll을 확인하거나 업로드합니다. TIA Portal이 설치되어 있으면 자동으로 감지됩니다.
